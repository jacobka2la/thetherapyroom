# The Therapy Room LLC

This is a Vercel-ready static website for The Therapy Room LLC.

## Deploy

Upload this folder to GitHub and import the repo into Vercel.
No build command is needed.

### Vercel settings
- Framework Preset: Other
- Build Command: leave blank
- Output Directory: leave blank
